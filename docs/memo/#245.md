### 1) User に `has_one_attached :avatar` を追加

`app/models/user.rb` に 1行追加します（関連の並びに合わせて `has_one :wallet` の近くが自然）。

**変更後（全文）**

```ruby
class User < ApplicationRecord
  devise :database_authenticatable, :recoverable, :rememberable, :validatable

  enum :role, { customer: 0, cast: 1, store_admin: 2, system_admin: 3 }

  validates :bio, length: { maximum: 500 }, allow_nil: true

  ROLE_LEVELS = {
    customer: 0,
    cast: 1,
    store_admin: 2,
    system_admin: 3
  }.freeze

  has_one_attached :avatar

  has_one :wallet, foreign_key: :customer_user_id, dependent: :destroy, inverse_of: :customer_user
  has_many :store_memberships, dependent: :destroy
  has_many :stores, through: :store_memberships
  has_many :favorite_booths, dependent: :destroy
  has_many :favorite_stores, dependent: :destroy

  # --- Soft delete ---
  def deleted?
    deleted_at.present?
  end

  # Devise: 停止ユーザーはログイン不可
  def active_for_authentication?
    super && !deleted?
  end

  def role_level
    ROLE_LEVELS.fetch(role.to_sym)
  rescue KeyError, NoMethodError
    -1
  end

  def at_least?(required_role)
    required_level = ROLE_LEVELS.fetch(required_role.to_sym)
    role_level >= required_level
  rescue KeyError
    false
  end

  def admin_of_store?(store_id)
    return false if store_id.blank?
    store_memberships.admin_only.exists?(store_id: store_id)
  end
end
```

### 2) ヘッダー表示（未添付フォールバック: “No Image” の小さい丸）

`app/views/layouts/application.html.erb` の、`display_name_or_email(current_user)` の前後に差し込みます。

あなたの希望どおり「メールアドレス（または名前）の前か後に小さく」なので、ここでは **名前の左側**に出します（視線誘導が自然）。

**差し替え対象ブロック（この部分だけ置換）**

```erb
            <% if user_signed_in? %>
              <span class="text-muted small">
                <%= display_name_or_email(current_user) %>
              </span>
              <%= role_badge(current_user) %>
```

**置換後**

```erb
            <% if user_signed_in? %>
              <span class="d-inline-flex align-items-center gap-2">
                <% if current_user.avatar.attached? %>
                  <%= image_tag current_user.avatar,
                        class: "rounded-circle border",
                        style: "width: 24px; height: 24px; object-fit: cover;",
                        alt: "avatar" %>
                <% else %>
                  <span
                    class="rounded-circle border text-muted d-inline-flex align-items-center justify-content-center"
                    style="width: 24px; height: 24px; font-size: 10px; line-height: 1;"
                    aria-label="No Image"
                    title="No Image">
                    No
                  </span>
                <% end %>

                <span class="text-muted small">
                  <%= display_name_or_email(current_user) %>
                </span>
              </span>

              <%= role_badge(current_user) %>
```

* 未添付でも `attached?` で分岐するのでエラーになりません
* “No Image（小さい丸）”は、丸の中に `No` を入れて分かりやすくしています（「No Image」全文は小さすぎるので）
