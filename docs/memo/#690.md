## これでどうなるか

* **通常画面**
  `--app-footer-base-h` を使って安定

* **standaloneスマホ**
  `env(safe-area-inset-bottom)` に加えて `12px` 余白追加

* **viewer / live**
  `--app-footer-h` は JS が触っても、今回の CSS 基準は `--app-footer-base-h` なので衝突しにくい

## ひとつ補足

`viewer-waiting-shell` にも今 `padding-bottom: var(--app-footer-h);` が入っていたので、
そこも standalone の見た目に合わせるなら、あとでこれに揃えたほうがよいです。

```scss
padding-bottom: calc(
  var(--app-footer-base-h) +
  env(safe-area-inset-bottom, 0px) +
  var(--safe-area-standalone)
);
```

ただ、まずは **通常画面の footer/main だけ直して見た目確認** でもいいです。

必要なら次に、**viewer と live も含めた修正箇所一覧** をそのままコピペできる形で出します。
