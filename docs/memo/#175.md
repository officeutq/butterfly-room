## 1) 実装方針（高レベル）

Issue #175 は **お気に入りの“基盤”のみ**なので、やることは最小で OK です。

* `favorite_booths` / `favorite_stores` を **別テーブル**で追加（polymorphic なし）
* `(user_id, booth_id)` / `(user_id, store_id)` を **DB UNIQUE index** で重複防止
* `created_at` は `t.timestamps` に含まれるので、そのまま一覧の並び（新しい順）に使える
* モデルは join モデルと同じく **presence の最低限**（unique は DB）
* 関連は `User -> favorite_*` を辿れるようにする
* テストは **UNIQUE が効いて例外が出る**ことを確認（最小）

### 4.3 関連の追記

#### `app/models/user.rb`（追記）

```ruby
  has_many :favorite_booths, dependent: :destroy
  has_many :favorite_stores, dependent: :destroy
```

* `User` は既に `wallet` や `store_memberships` を `dependent: :destroy` しているので、favorite も揃えて `:destroy` にしています。

#### `app/models/booth.rb`（追記）

```ruby
  has_many :favorite_booths, dependent: :destroy
```

#### `app/models/store.rb`（追記）

```ruby
  has_many :favorite_stores, dependent: :destroy
```

* `Booth` は `stream_sessions` / `booth_casts` を `restrict_with_error` にしていますが、favorite は業務的に「削除を止める」ほど重要ではないので、掃除される `:destroy` の方が事故りにくい（削除が通らず詰まるのを防ぐ）という判断です。
* `Store` も他の関連が `dependent: :destroy` 中心なので合わせています。

### 4.4 テスト（最小：DB UNIQUE が効く）

#### `test/models/favorite_booth_test.rb`

```ruby
# frozen_string_literal: true

require "test_helper"

class FavoriteBoothTest < ActiveSupport::TestCase
  test "user_id + booth_id is unique at DB level" do
    store = Store.create!(name: "store")
    booth = Booth.create!(store: store, name: "booth", status: :offline)
    user  = User.create!(email: "u1@example.com", password: "password", role: :customer)

    FavoriteBooth.create!(user: user, booth: booth)

    assert_raises(ActiveRecord::RecordNotUnique) do
      FavoriteBooth.create!(user: user, booth: booth)
    end
  end

  test "user can access favorite_booths" do
    store = Store.create!(name: "store")
    booth = Booth.create!(store: store, name: "booth", status: :offline)
    user  = User.create!(email: "u2@example.com", password: "password", role: :customer)

    FavoriteBooth.create!(user: user, booth: booth)

    assert_equal 1, user.favorite_booths.count
    assert_equal booth.id, user.favorite_booths.first.booth_id
  end
end
```
