## 反映後のイメージ

該当部分だけ抜くと、こうです。

```erb
<%= form_for(resource, as: resource_name, url: session_path(resource_name)) do |f| %>
  <div class="mb-3">
    <%= f.label :email, class: "form-label" %>
    <%= f.email_field :email, autofocus: true, autocomplete: "email", class: "form-control" %>
  </div>

  <div class="mb-3">
    <%= f.label :password, class: "form-label" %>
    <%= f.password_field :password, autocomplete: "current-password", class: "form-control" %>
  </div>

  <% if devise_mapping.rememberable? %>
    <div class="form-check mb-3">
      <%= f.check_box :remember_me, class: "form-check-input" %>
      <%= f.label :remember_me, class: "form-check-label" %>
    </div>
  <% end %>

  <div class="mb-3">
    <%= link_to "パスワードを忘れましたか？", new_user_password_path, class: "link-secondary" %>
  </div>

  <div class="d-flex gap-2 mb-3">
    <%= f.submit "ログイン", class: "btn btn-primary" %>
    <%= link_to "ホーム", root_path, class: "btn btn-outline-secondary" %>
  </div>
<% end %>
```

## 動作確認

1. ログイン画面を開く
2. 「パスワードを忘れましたか？」リンクを押す
3. Devise のメールアドレス入力画面に遷移する
4. 既存ユーザーのメールアドレスを入れて送信する
5. development なら `/letter_opener`、production なら実メールで確認する
6. メール内リンクから新しいパスワードを設定できることを確認する
