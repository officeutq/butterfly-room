今回の変更ポイントはこれです。

* `booth.booth_casts.max_by` をやめて `booth.primary_cast_user` に統一
* `booth_status_badge` を waiting 画面から削除
* 「所属キャスト」「所属店舗」ラベルを削除
* `viewer-waiting-panel` / `viewer-waiting-info-card` 系のカード前提UIを廃止
* ブース説明は最下部へ移動し、未設定時はブロックごと非表示
* `data-viewer-page-target="boothFavoriteAnchor"` と `storeFavoriteAnchor` は維持
* waiting 画面だけ縦スクロール可能に変更

確認ポイントはこのあたりです。

* standby で waiting レイアウトになる
* live / away では既存UIのまま切り替わる
* サムネイルの上に余白が出ていない
* 説明が長いときに下まで自然にスクロールできる
* favorite ボタンがブース名右、店舗名右に出る
