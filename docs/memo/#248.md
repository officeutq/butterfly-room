* `feature/248-users-show`

## 1) routes 追加

`config/routes.rb` の「Common profile」の近く（dashboard/profileの直後あたり）に追加：

```ruby
  # --- Public user profiles (login required) ---
  resources :users, only: %i[show]
```

※ `namespace :system_admin do ... resources :users ... end` とは別パス（`/system_admin/users`）なので衝突しません。

---

## 2) Controller 追加

新規作成：`app/controllers/users_controller.rb`

```ruby
# frozen_string_literal: true

class UsersController < ApplicationController
  before_action :authenticate_user!

  def show
    @user = User.find(params[:id])
  end
end
```

* 存在しないIDは `ActiveRecord::RecordNotFound` で 404 になります（Done条件OK）

---

## 3) view 追加

新規作成：`app/views/users/show.html.erb`

```erb
<div class="container mt-4" style="max-width: 960px;">
  <% display_name = @user.display_name.presence || "ななしさん" %>
  <% content_for :title, display_name %>

  <div class="d-flex justify-content-between align-items-start gap-3">
    <div>
      <h1 class="mb-1"><%= display_name %></h1>
      <p class="text-muted mb-0">ユーザープロフィール</p>
    </div>
  </div>

  <hr class="my-4">

  <div class="mb-4">
    <% if @user.avatar.attached? %>
      <div class="mb-3">
        <%= image_tag @user.avatar,
              class: "border rounded",
              style: "width: 200px; height: 200px; object-fit: cover;",
              alt: "avatar" %>
      </div>
    <% else %>
      <div
        class="border rounded d-flex align-items-center justify-content-center text-muted"
        style="width: 200px; height: 200px; background: #f8f9fa;">
        No Image
      </div>
    <% end %>
  </div>

  <% if @user.bio.present? %>
    <h2 class="h5 mb-3">自己紹介</h2>
    <div class="mb-4">
      <div class="border rounded p-3 bg-white" style="white-space: pre-wrap;">
        <%= @user.bio %>
      </div>
    </div>
  <% end %>

  <div class="mt-4">
    <%= link_to "戻る", dashboard_path, class: "btn btn-outline-secondary" %>
  </div>
</div>
```

ポイント：

* email は表示しません
* bio は「空欄でOK」なので、**未入力ならセクションごと非表示**（落ちない・UIもすっきり）
* avatar フォールバックは 200x200 のグレー枠＋ No Image
* `white-space: pre-wrap;` で改行を保持（textarea入力と相性良い）
