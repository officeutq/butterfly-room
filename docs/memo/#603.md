## 1. `app/views/home/show.html.erb`

ドロップダウン部分を、`btn-check` ベースのトグルに置き換えます。
`mode` は hidden ではなく、ラジオの `name="mode"` でそのまま送ります。

```erb
<% if user_signed_in? %>
  <%# --- Search (Input group / GET) --- %>
  <div class="mb-3">
    <%= form_with url: root_path, method: :get, scope: nil do |f| %>
      <div class="input-group">
        <div class="btn-group" role="group" aria-label="表示切り替え">
          <input
            type="radio"
            class="btn-check"
            name="mode"
            id="home_mode_booths"
            value="booths"
            autocomplete="off"
            <%= "checked" if @mode != "stores" %>>
          <label class="btn btn-outline-secondary" for="home_mode_booths">ブース</label>

          <input
            type="radio"
            class="btn-check"
            name="mode"
            id="home_mode_stores"
            value="stores"
            autocomplete="off"
            <%= "checked" if @mode == "stores" %>>
          <label class="btn btn-outline-secondary" for="home_mode_stores">店舗</label>
        </div>

        <%= f.text_field :q,
              value: @q,
              class: "form-control",
              placeholder: "ブース名 / ストア名で検索（部分一致）",
              autocomplete: "off" %>

        <button class="btn btn-primary" type="submit" aria-label="検索">
          <i class="bi bi-search"></i>
        </button>
      </div>
    <% end %>
  </div>

  <%# --- Results (cards) --- %>
  <% if @mode == "stores" %>
    <% if @stores.present? %>
      <div class="row row-cols-1 row-cols-md-3 g-3">
        <% @stores.each do |store| %>
          <% favorited = @favorite_store_ids.include?(store.id) %>

          <div class="col">
            <%= render "stores/card", store: store, favorited: favorited %>
          </div>
        <% end %>
      </div>
    <% else %>
      <div class="alert alert-secondary" role="alert">
        表示できる店舗がありません。
      </div>
    <% end %>
  <% else %>
    <% if @booths.present? %>
      <div class="row row-cols-1 row-cols-md-3 g-3">
        <% @booths.each do |booth| %>
          <div class="col">
            <%= render "booths/card",
                  booth: booth,
                  booth_favorited: @favorite_booth_ids.include?(booth.id),
                  store_favorited: @favorite_store_ids.include?(booth.store_id) %>
          </div>
        <% end %>
      </div>
    <% else %>
      <div class="alert alert-secondary" role="alert">
        表示できるブースがありません。
      </div>
    <% end %>
  <% end %>
<% else %>
  <div style="max-width: 720px;">
    <h1 class="mb-3">ようこそ</h1>
    <p class="text-muted">視聴するにはログイン、または新規アカウント作成を行ってください。</p>

    <div class="d-flex gap-2 mt-3">
      <%= link_to "ログイン", new_user_session_path, class: "btn btn-primary" %>
      <%= link_to "新規アカウント作成", sign_up_path, class: "btn btn-outline-primary" %>
    </div>
  </div>
<% end %>
```

## 2. `app/views/favorites/booths/index.html.erb`

右上の単発リンクをやめて、見出し下にトグルを置きます。

```erb

  <div class="d-flex flex-column gap-3">

            "aria-current": "page" %>

      <%= link_to "店舗",
            favorites_stores_path,
            class: "btn btn-outline-secondary" %>
    </div>
  </div>

  <hr class="my-4">

  <% if @favorite_booths.blank? %>
    <div class="alert alert-secondary" role="alert">
      お気に入りブースはありません。
    </div>
  <% else %>
    <div class="row row-cols-1 row-cols-md-3 g-3">
      <% @favorite_booths.each do |favorite| %>

        <div class="col">
          <%= render "booths/card",
                booth: booth,
                booth_favorited: true,
                store_favorited: @favorite_store_ids.include?(booth.store_id) %>
        </div>
      <% end %>
    </div>
  <% end %>
</div>
```

## 補足

今回の差分だと、

* `HomeController`
* `Favorites::BoothsController`
* `Favorites::StoresController`

は **変更不要** です。
既存の `mode` と既存ルートをそのまま使えます 

---

## この実装の意図

* home は検索UIの一部なので、**input-group 内に収まる形**
* favorites は検索がないので、**単独の btn-group**
* favorites はページ遷移のままなので、**安全に既存維持**
* Bootstrap 標準寄りなので、余計な CSS なしでもまず成立

---

## 確認ポイント

実装後はこのあたりを見るといいです。

* home で「ブース / 店舗」を切り替えて検索できるか
* `q` を入れたまま mode 切替後に submit して意図通りか
* favorites で active 側の見た目が分かりやすいか
* スマホ幅でトグルが折れずに見えるか

必要なら次に、**見た目を少し整える最小SCSS差分** だけ追加で出します。
