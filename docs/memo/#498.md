Issue文では `store_admin / system_admin → /admin/booths` だったので、そのままにしています。

つまり今回の実装は **Issue優先** です。

## 2. 既存の「キャスト管理」「ブースにキャストを紐づける」は消している件

Issue #498 の表示カード定義に入っていないので出していません。
これは #502 や #499 に責務を寄せる流れとも整合しています。
