## 3. 変更

`app/views/booths/_stream_state.html.erb`

live / away 中のみ、Overlay 左下へコメント panel を出します。
`viewer-legacy-offscreen` にあった既存コメント DOM とフォームは、この Issue では Overlay に移すので削除します。

```erb
<%# locals: booth:, stream_session:, comments:, drink_items: %>

<% live_like = booth.live? || booth.away? %>

<div class="viewer-stream-state">
  <% if stream_session.present? && live_like %>
    <%= turbo_stream_from [stream_session, :pending_drink_orders] %>
    <%= turbo_stream_from [stream_session, :comments] %>
    <%= turbo_stream_from [stream_session, :viewer_stage] %>

    <% content_for :viewer_overlay_comment do %>
      <%= render "booths/comment_section",
            booth: booth,
            stream_session: stream_session,
            comments: comments %>
    <% end %>

    <div class="viewer-stage-shell">
      <div id="viewer_stage">
        <%= render "stream_sessions/ivs_viewer", stream_session: stream_session %>

        <div id="viewer_flash_layer">
          <div id="flash_inner">
            <% if notice.present? %>
              <%= render "shared/flash_message", level: "success", message: notice %>
            <% end %>

            <% if alert.present? %>
              <%= render "shared/flash_message", level: "danger", message: alert %>
            <% end %>
          </div>
        </div>
      </div>
    </div>

    <div class="viewer-legacy-offscreen" aria-hidden="true">
      <div
        data-controller="presence-poll"
        data-presence-poll-ping-url-value="<%= ping_stream_session_presence_path(stream_session) %>"
        data-presence-poll-summary-url-value="<%= presence_summary_stream_session_path(stream_session) %>"
        data-presence-poll-interval-ms-value="20000"
      >
        <% title_display = stream_session.title.presence || booth.name %>

        <div class="mb-2">
          <div class="fw-semibold"><%= title_display %></div>
          <% if stream_session.description.present? %>
            <div class="text-muted small"><%= stream_session.description %></div>
          <% end %>
        </div>

        <hr>

        viewers: <span data-presence-poll-target="count">-</span>
      </div>

      <hr>

      <%= render "booths/drink_menu",
            booth: booth,
            stream_session: stream_session,

      <hr>

      <%= render "stream_sessions/pending_drink_orders", stream_session: stream_session %>

    <div class="viewer-stage-shell viewer-stage-shell-empty">
      <div id="viewer_stage">
        <div id="viewer_flash_layer">
          <div id="flash_inner">
            <% if notice.present? %>

            <% end %>

            <% if alert.present? %>
              <%= render "shared/flash_message", level: "danger", message: alert %>
            <% end %>
          </div>
        </div>
      </div>
    </div>

      <div class="text-muted">現在配信していません</div>

            booth: booth,

            drink_items: drink_items %>

      <div class="text-muted">配信中のみ表示されます</div>
```

## 補足

今回の実装は `content_for :viewer_overlay_comment` を使って `ivs_viewer` 内の Overlay スロットへ差し込む形にしています。
これで Overlay 枠の定義は `stream_sessions/_ivs_viewer.html.erb` に残しつつ、表示内容の責務は `booths/_stream_state.html.erb` 側に置けます。
