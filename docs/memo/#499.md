* `Admin::BoothsController#index`

  * `@current_booth_id = session[:current_booth_id]`
  * `@cast_memberships` を `Admin::BoothCastsController#index` 相当で追加

* `app/assets/stylesheets/components/_booths_index.scss`

## 1. `app/controllers/admin/booths_controller.rb`

`index` だけ差し替えでOKです。

```ruby
def index
  @include_archived = ActiveModel::Type::Boolean.new.cast(params[:archived])

  scope =
    current_store
      .booths
      .includes(:thumbnail_image_attachment, booth_casts: :cast_user)

  scope = scope.active unless @include_archived

  @booths = scope.order(id: :desc)
  @current_booth_id = session[:current_booth_id]

  @cast_memberships =
    StoreMembership
      .includes(:user)
      .where(store_id: current_store.id, membership_role: :cast)
      .order(:id)
end
```

## 補足

今回の実装では、紐づけ submit 後の戻り先は既存 `Admin::BoothCastsController#create` のままなので、**送信後は `/admin/booth_casts` に飛びます**。
Issue 本文の「既存 create をそのまま使ってよい」に沿うならまずはこれでOKですが、**`/admin/booths` に戻したいなら `Admin::BoothCastsController#create` の redirect だけ後で寄せる**のがよいです。

その場合は次のように変えるのが自然です。

* 成功時: `redirect_to admin_booths_path, notice: "..."`
* 失敗時: `redirect_to admin_booths_path, alert: "..."`
* `archived=1` の維持までやるなら hidden で戻り先を渡す

まずは上の3ファイルだけで #499 の主要求はかなり満たせます。
必要なら次に、**`Admin::BoothCastsController#create` を `/admin/booths` 戻りへ寄せた版**もそのまま貼れる形で出します。
