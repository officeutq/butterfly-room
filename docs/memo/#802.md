# ❌ 今のバグの正体

これです👇

```js
this._banubaVideoTrack
this._banubaStageStream
this._publishedVideoSource = "banuba"
```

👉 DeepAR使ってるのに **Banuba前提でpublishしてる**

---

さらに致命的なのがここ👇

```js
this._banubaStageStream = this._banubaVideoTrack ? new LocalStageStream(this._banubaVideoTrack) : null
```

👉 DeepARのtrackが完全に無視されてる

---

# ❗ エラーの原因（これ）

```text
映像状態の同期が必要です
```

これは `ui_state.js` のこれ👇

```js
if (ctx._broadcasting && ctx._publishedVideoSource !== "banuba")
```

👉 DeepARのとき `"deepar"` になるので必ずエラー

---

# 🔥 修正方針（これが本質）

```text
banuba を消す
```

ではなく

```text
provider に統一する
```
