今回の変更は `app/views/booths/_drink_menu.html.erb` のみで足ります。

やることは次の1点です。

* `f.submit "#{item.name}(#{item.price_points}pt)"` をやめて、`type="submit"` の `button` の中に `drink_item_display(item)` を描画する

これで Issue #536 の要件どおり、

* hidden field は維持
* 利用可否分岐は維持
* turbo-frame 構造は維持

## 修正後ファイル全文

`app/views/booths/_drink_menu.html.erb`

```erb
<%# locals: booth:, stream_session:, drink_items:, can_create_drink_order: %>

<turbo-frame id="viewer_drink_menu">
  <% live_like = booth.live? || booth.away? %>
  <% has_drink_items = Array(drink_items).any? %>

  <% if !live_like %>
    <div class="small text-white-50">配信中のみドリンクを送れます。</div>
  <% elsif !has_drink_items %>
    <div class="small text-white-50">現在送れるドリンクがありません。</div>
  <% elsif !can_create_drink_order %>
    <div class="small text-white-50">現在このアカウントではドリンク送信を利用できません。</div>
  <% else %>
    <div class="viewer-drink-menu-list" role="group" aria-label="ドリンク送信">
      <% Array(drink_items).each do |item| %>
        <%= form_with url: stream_session_drink_orders_path(stream_session),
                      scope: :drink_order,
                      method: :post,
                      class: "viewer-drink-menu-form" do |f| %>
          <%= f.hidden_field :drink_item_id, value: item.id %>
          <button type="submit" class="viewer-drink-menu-btn">
            <%= drink_item_display(item) %>
          </button>
        <% end %>
      <% end %>
    </div>
  <% end %>
</turbo-frame>
```
