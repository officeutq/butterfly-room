実装方針はこれで進めるのが最小変更で安全です。

* `application.html.erb` の `body` に、配信画面のときだけ `cast-live-layout` を付与する
* `_header.html.erb` の `sticky-top` は残したままにし、`body.cast-live-layout #app_header` で fixed overlay に上書きする
* `_live.scss` に header 用の live 専用スタイルを追加し、半透明背景・z-index・幅を制御する
* 他画面は `cast-live-layout` が付かないので影響なし

この形なら、Issue #396 の範囲に収まりつつ、#394 / #395 の safe area 設計とも噛み合います。
