## 2) Controller 作成

新規作成：`app/controllers/profiles_controller.rb`

```ruby
# frozen_string_literal: true

class ProfilesController < ApplicationController
  before_action :authenticate_user!

  def edit
    @user = current_user
  end

  def update
    @user = current_user

    if @user.update(profile_params)
      redirect_to edit_profile_path, notice: "プロフィールを更新しました"
    else
      render :edit, status: :unprocessable_entity
    end
  end

  private

  def profile_params
    params.require(:user).permit(:display_name, :bio, :avatar)
  end
end
```

* 更新対象は常に `current_user`（id を受け取らない）
* 成功後は `/profile/edit` に戻す（希望どおり）

## 3) view 作成（form_with）

新規作成：`app/views/profiles/edit.html.erb`

```erb
<h1>プロフィール編集</h1>

<%= form_with model: @user, url: profile_path do |f| %>
  <% if @user.errors.any? %>
    <div class="alert alert-danger" role="alert">
      <ul class="mb-0">
        <% @user.errors.full_messages.each do |msg| %>
          <li><%= msg %></li>
        <% end %>
      </ul>
    </div>
  <% end %>

  <div class="mb-3">
    <%= f.label :display_name, "表示名", class: "form-label" %>
    <%= f.text_field :display_name, class: "form-control" %>
    <div class="form-text">未入力の場合は email が表示されます（将来変更の可能性あり）</div>
  </div>

  <div class="mb-3">
    <%= f.label :bio, "自己紹介（bio）", class: "form-label" %>
    <%= f.text_area :bio, class: "form-control", rows: 6 %>
    <div class="form-text">最大500文字</div>
  </div>

  <div class="mb-3">
    <%= f.label :avatar, "プロフィール画像（avatar）", class: "form-label" %>

    <% if @user.avatar.attached? %>
      <div class="mb-2">
        <%= image_tag @user.avatar, style: "max-width: 160px; height: auto;" %>
      </div>
      <div class="text-muted small mb-2">
        画像を差し替える場合は新しいファイルを選択してください（削除はPhase1後回し）
      </div>
    <% end %>

    <%= f.file_field :avatar, class: "form-control" %>
    <div class="form-text">png / jpg / webp</div>
  </div>

  <div class="d-flex gap-2">
    <%= f.submit "更新する", class: "btn btn-primary" %>
    <%= link_to "戻る", dashboard_path, class: "btn btn-outline-secondary" %>
  </div>
<% end %>
```

* stores/edit のパターンを踏襲（alert + ul）
* avatar は「差し替えのみ」の文言を合わせる
