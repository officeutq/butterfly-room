了解です。
#354 は、既存の viewer Overlay 基盤に **ドリンク表示パネルだけを差し込む** 形で、最小変更で実装できます。

実装の狙いは次の通りです。

* live / away 時だけ、右上 slot にドリンク一覧を表示
* 一覧更新は既存の `pending_drink_orders` Turbo Stream をそのまま利用
* viewer では操作ボタンなし
* パネル内だけ縦スクロールし、ページ全体はスクロールさせない
* 見た目は cast/live の `live-drink-panel` に寄せる

変更対象はこの3ファイルです。

* `app/views/booths/_stream_state.html.erb`
* `app/views/stream_sessions/_ivs_viewer.html.erb`
* `app/assets/stylesheets/components/_viewer.scss`

以下、そのまま貼れる完成形です。
