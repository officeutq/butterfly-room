今回は **`ApplicationHelper` には追加せず**、`BuildHelper` を新設する方針でいくのがきれいです。

実装方針はこれです。

1. `app/helpers/build_helper.rb` を新規追加
2. `config/app_build.yml` から現在環境の `build_number` を読む helper を定義
3. `_header.html.erb` の区切り線の下、ログアウトの上に `Build 2` 形式で表示
4. 未ログイン時は既存どおり何も表示しない

貼り付け用コードです。

### 1. 新規追加: `app/helpers/build_helper.rb`

```ruby
# frozen_string_literal: true

module BuildHelper
  def current_build_number
    Rails.application.config_for(:app_build).fetch("build_number")
  end

  def current_build_label
    "Build #{current_build_number}"
  end
end
```

### 2. 置き換え: `app/views/layouts/_header.html.erb`


* 今回は表示だけなので、テストなし運用でも十分小さくまとまっています
