## 最終実装プラン（タスク順）

1. **DB**

* `stream_sessions` に `title:string` / `description:text` 追加
* `StreamSession` に最大長 validation（title 64 / description 256）

2. **更新エンドポイント（cast）**

* `PATCH /cast/stream_sessions/:id/metadata`
* 認可：`StartService#authorize!` と同等条件（system_admin / store_admin=自store / cast=booth_casts）
* 必須条件：`booth.current_stream_session_id == stream_session.id`
* 状態ガード：`booth.status == standby` 以外は **409**
* 成功時：DB更新＋ `StreamSessionNotifier.broadcast_stream_state(booth:)` で viewer に即反映
* Turbo Stream：`cast_stream_meta` を replace ＋ `flash_inner` に成功メッセージ append

3. **cast UI（live画面）**

* `standby` のときだけ編集フォームを表示（現在値を初期表示）
* `live/away` は表示専用（編集UIなし）
* `standby→live` の瞬間：既存 `Cast::BoothsController#status` の turbo_stream で `cast_stream_meta` も replace してフォーム→表示専用に切替

4. **viewer UI（booths/_stream_state）**

* タイトル・説明を表示（title blank は表示時に `booth.name` を使う、description blank は非表示）
* `broadcast_stream_state` でリロード不要の自動反映に乗る

5. **テスト**

* まずは model の length validation
* 次に request/integration で「403 / 409」ガード（※User/Fixture事情があるので、あなたの既存fixture名に合わせて微調整が入る可能性があります）

## 4) cast 更新処理（controller）

`app/controllers/cast/stream_sessions_controller.rb`

```ruby
# frozen_string_literal: true

module Cast
  class StreamSessionsController < Cast::BaseController
    def finish
      session = StreamSession.find(params[:id])

      StreamSessions::EndService.new(
        stream_session: session,
        actor: current_user
      ).call

      redirect_to cast_booth_path(session.booth_id), notice: "配信終了"
    rescue => e
      redirect_to cast_booth_path(session.booth_id), alert: e.message
    end

    def pending_drink_orders
      stream_session = StreamSession.find(params[:id])

      render partial: "cast/stream_sessions/pending_drink_orders",
             locals: { stream_session: stream_session }
    end

    # PATCH /cast/stream_sessions/:id/metadata
    def metadata
      stream_session = StreamSession.find(params[:id])
      booth = stream_session.booth

      # 必須: current session 一致
      unless booth.current_stream_session_id == stream_session.id
        return head :forbidden
      end

      # 認可: StartService と同等
      unless operable_booth_for_stream_session?(booth)
        return head :forbidden
      end

      # 状態ガード: standby のみ許可
      unless booth.standby?
        return respond_conflict("スタンバイ中のみ編集できます")
      end

      stream_session.update!(metadata_params)

      # viewer 側に即反映（booth:stream_state 購読に乗せる）
      StreamSessionNotifier.broadcast_stream_state(booth: booth)

      respond_to do |format|
        format.html do
          redirect_to live_cast_booth_path(booth), notice: "設定しました"
        end

        format.turbo_stream do
          render turbo_stream: [
            turbo_stream.replace(
              "cast_stream_meta",
              partial: "cast/booths/stream_meta",
              locals: { booth: booth, stream_session: stream_session }
            ),
            turbo_stream.append(
              "flash_inner",
              partial: "shared/flash_message",
              locals: { level: "success", message: "設定しました" }
            )
          ]
        end

        format.json { render json: { ok: true }, status: :ok }
        format.any { head :ok }
      end
    rescue ActiveRecord::RecordInvalid => e
      message = e.record.errors.full_messages.join(", ").presence || "入力に誤りがあります"

      respond_to do |format|
        format.html { redirect_to live_cast_booth_path(booth), alert: message }
        format.turbo_stream do
          render turbo_stream: turbo_stream.append(
            "flash_inner",
            partial: "shared/flash_message",
            locals: { level: "danger", message: message }
          ), status: :unprocessable_entity
        end
        format.json { render json: { error: message }, status: :unprocessable_entity }
        format.any { render plain: message, status: :unprocessable_entity }
      end
    rescue ActiveRecord::RecordNotFound
      head :not_found
    end

    private

    def metadata_params
      params.require(:stream_session).permit(:title, :description)
    end

    def operable_booth_for_stream_session?(booth)
      actor = current_user

      return false if actor.blank?
      return true if actor.system_admin?

      if actor.at_least?(:store_admin)
        return actor.admin_of_store?(booth.store_id)
      end

      BoothCast.exists?(cast_user_id: actor.id, booth_id: booth.id)
    end

    def respond_conflict(message)
      respond_to do |format|
        format.html { redirect_to live_cast_booth_path(booth), alert: message }
        format.turbo_stream do
          render turbo_stream: turbo_stream.append(
            "flash_inner",
            partial: "shared/flash_message",
            locals: { level: "danger", message: message }
          ), status: :conflict
        end
        format.json { render json: { error: message }, status: :conflict }
        format.any { render plain: message, status: :conflict }
      end
    end
  end
end
```

> 注意：`respond_conflict` 内で `booth` 参照してしまうのを避けるため、必要なら引数で `booth` を渡す形に変えてください（下の「安全版」を推奨）。

**安全版（推奨）**：`respond_conflict(booth, message)` にする差分だけ反映してください。

```ruby
def respond_conflict(booth, message)
  respond_to do |format|
    format.html { redirect_to live_cast_booth_path(booth), alert: message }
    format.turbo_stream do
      render turbo_stream: turbo_stream.append(
        "flash_inner",
        partial: "shared/flash_message",
        locals: { level: "danger", message: message }
      ), status: :conflict
    end
    format.json { render json: { error: message }, status: :conflict }
    format.any { render plain: message, status: :conflict }
  end
end
```

そして呼び出しを：

```ruby
return respond_conflict(booth, "スタンバイ中のみ編集できます")
```

にしてください。

## 7) standby→live の瞬間にフォームを消す（status turbo_stream を拡張）

`app/controllers/cast/booths_controller.rb` の `def status` の `format.turbo_stream do` を **2つreplace**にします。

現状：

```ruby
format.turbo_stream do
  render turbo_stream: turbo_stream.replace(
    "cast_comment_section",
    partial: "cast/booths/comment_section",
    locals: { booth: booth, stream_session: stream_session, comments: comments }
  )
end
```

変更後：

```ruby
format.turbo_stream do
  render turbo_stream: [
    turbo_stream.replace(
      "cast_stream_meta",
      partial: "cast/booths/stream_meta",
      locals: { booth: booth, stream_session: stream_session }
    ),
    turbo_stream.replace(
      "cast_comment_section",
      partial: "cast/booths/comment_section",
      locals: { booth: booth, stream_session: stream_session, comments: comments }
    )
  ]
end
```

これで `standby -> live` の瞬間、**booth.status が live になった Turbo 更新**でフォームが表示専用に切り替わります。
