## 実装プラン（最小差分・Issue #297 準拠）

1. **`app/views/layouts/application.html.erb`**

   * 共通Footer中央の「配信」リンク部分に、`cast/booths#show` のときだけ分岐を追加
   * その分岐内で、**`app/views/cast/booths/show.html.erb` にあった “スタンバイ開始” ボタンと同一のHTML/挙動**を出力
   * `archived?` のときは show と同じく **disabled ボタン**を出力
   * それ以外の画面は従来通り「配信」リンク

2. **`app/views/cast/booths/show.html.erb`**

   * 既存の「スタンバイ開始」ボタンを削除（表示場所のみ移動）

### デグレ影響範囲（Epic #285 ルール準拠）

* 配信状態遷移への影響：なし（StartService/遷移先は現状のまま）

## 変更コード（そのままコピペ可）

### 1) `app/views/layouts/application.html.erb`

**置換対象**：Footer中央 `<div>` 内のこの部分

```erb
            <div>
              <% if user_signed_in? && current_user.at_least?(:cast) %>
                <% stream_path =
                     if current_user.cast?
                       booth = layout_current_booth
                       if booth.present?
                         booth.offline? ? cast_booth_path(booth) : live_cast_booth_path(booth)
                       else
                         cast_booths_path
                       end
                     else
                       cast_booths_path
                     end %>

                <%= link_to "配信", stream_path, class: "btn btn-sm btn-primary" %>
              <% end %>
            </div>
```

**上記を、以下に丸ごと置き換え**：

```erb
            <div>
              <% if user_signed_in? && current_user.at_least?(:cast) %>
                <% cast_booths_show = (controller_path == "cast/booths" && action_name == "show") %>

                <% if cast_booths_show %>
                  <% booth = @booth %>

                  <% if booth.archived? %>
                    <button class="btn btn-primary" disabled>スタンバイ開始</button>
                  <% else %>
                    <%= button_to "スタンバイ開始",
                          cast_booth_stream_sessions_path(booth),
                          method: :post,
                          class: "btn btn-primary" %>
                  <% end %>
                <% else %>
                  <% stream_path =
                       if current_user.cast?
                         booth = layout_current_booth
                         if booth.present?
                           booth.offline? ? cast_booth_path(booth) : live_cast_booth_path(booth)
                         else
                           cast_booths_path
                         end
                       else
                         cast_booths_path
                       end %>

                  <%= link_to "配信", stream_path, class: "btn btn-sm btn-primary" %>
                <% end %>
              <% end %>
            </div>
```

ポイント：

* `controller_path/action_name` 条件は Issue 指定どおり
* ボタンは **show内にあったものと同一**（文言 / class / method / path / disabled）
* live は `live_layout_moved` でそもそもこのFooterが描画されないので影響しません

### 2) `app/views/cast/booths/show.html.erb`

**削除対象**：このブロックを丸ごと削除

```erb
<% if @booth.archived? %>
  <button class="btn btn-primary" disabled>スタンバイ開始</button>
<% else %>
  <%= button_to "スタンバイ開始",
        cast_booth_stream_sessions_path(@booth),
        method: :post,
        class: "btn btn-primary" %>
<% end %>

<hr>
```

削除後は、直後の「戻る/home」リンクがそのまま表示されます（必要なら `<hr>` を残すのはOKですが、Issue要件は「ボタン削除」なので最小差分としては上のブロックごと消すのが一番安全です）。
