## 実装プラン（順序つき）

1. **ヘルパ追加**

* `ApplicationHelper` に `display_name_or_anonymous(user)` を追加

  * `user.blank?` → `""`
  * `user.display_name.presence || "ななしさん"`

2. **一括置換（洗い出し済み箇所のみ）**

* `display_name_or_email(` の呼び出しを **すべて** `display_name_or_anonymous(` に置換
  （ただし “emailを表示したい箇所” は `user.email` 明示表示＝今回すでに layout の dropdown 内で実現済み）

3. **メール露出の副作用も潰す（avatar fallback）**

* `user_avatar_badge` の fallback で `label/title` に email が入らないようにする（＝匿名ヘルパへ置換）

4. **テスト実行**

* 既存テストが通ることを確認
