今回は **`app/views/system_admin/users/index.html.erb` を全文置き換え** で進めて大丈夫です。

以下に差し替えてください。

```erb
<h1>ユーザー管理</h1>

<div class="mb-3 d-flex gap-2">
  <%= link_to "新規作成", new_system_admin_user_path, class: "btn btn-primary" %>
</div>

<% if @users.blank? %>
  <div class="alert alert-warning" role="alert">
    表示できるユーザーがありません。
  </div>
<% else %>
  <div class="referral-codes-list">
    <% @users.each do |u| %>
      <div class="card referral-code-card <%= "text-muted" if u.deleted? %>">
        <div class="card-body">
          <div class="referral-code-row">
            <div class="referral-code-label">名前</div>
            <div class="referral-code-value d-flex align-items-center gap-2">
              <%= user_avatar_badge(u, size: 28) %>
              <span class="fw-semibold"><%= display_name_or_anonymous(u) %></span>
            </div>
          </div>

          <div class="referral-code-row">
            <div class="referral-code-label">Email</div>
            <div class="referral-code-value fw-semibold text-break"><%= u.email %></div>
          </div>

          <div class="referral-code-row">
            <div class="referral-code-label">Role</div>
            <div class="referral-code-value">
              <%= role_badge(u) %>
            </div>
          </div>

          <div class="referral-code-row">
            <div class="referral-code-label">Created</div>
            <div class="referral-code-value text-muted small"><%= l(u.created_at) %></div>
          </div>

          <div class="referral-code-row">
            <div class="referral-code-label">Status</div>
            <div class="referral-code-value">
              <% if u.deleted? %>
                <span class="badge text-bg-secondary">停止中</span>
              <% else %>
                <span class="badge text-bg-success">稼働中</span>
              <% end %>
            </div>
          </div>

          <div class="pt-3">
            <div class="referral-code-inline-actions justify-content-end">
              <%= link_to "編集", edit_system_admin_user_path(u), class: "btn btn-outline-secondary btn-sm" %>

              <% unless u.deleted? %>
                <%= button_to "停止",
                      system_admin_user_path(u),
                      method: :delete,
                      class: "btn btn-outline-danger btn-sm",
                      data: { turbo_confirm: "このユーザーを停止します。よろしいですか？" } %>
              <% end %>
            </div>
          </div>
        </div>
      </div>
    <% end %>
  </div>
<% end %>
```

今回の実装でやっていることはこのあたりです。

* テーブルを廃止してカード一覧化
* 「ダッシュボードへ戻る」ボタンを削除
* 名前項目を追加
* アバター + 名前表示に変更
* Email はそのまま表示
* Role は英語のまま `role_badge` で表示
* Status は日本語化
* 停止済みユーザーは `text-muted` を維持
* 0件時の空状態を追加

確認ポイントはこのあたりです。

* 一覧がカード表示になる
* 名前がアバター付きで見える
* Email が表示される
* Role が badge で表示される
* Status が「稼働中 / 停止中」になる
* 「ダッシュボードへ戻る」ボタンが消えている
* 停止済みユーザーが薄く見える

もし見た目をさらに詰めるなら、次は「Created の見え方」か「Role badge の色味」くらいです。
