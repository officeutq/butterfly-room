了解です。
discard 後の前提で、そのまま貼れる形を出します。

今回は既存流儀に合わせて、

* partial は `comment` だけで安全に描画できるようにする
* cast / viewer の出し分けは `body.cast-live-layout` / `body.viewer-layout` で CSS 側に寄せる
* hidden 状態はまだ実データが無いので、**将来 `.is-hidden-comment` を付けたら切り替わる土台だけ先に入れる**

で進めます。

修正ファイルはこの2つです。

* `app/views/comments/_comment.html.erb`
* `app/assets/stylesheets/components/_broadcast_comment.scss`

## app/views/comments/_comment.html.erb

```erb
<div id="comment_<%= comment.id %>" class="comment comment-item">
  <% case comment.kind %>
  <% when Comment::KIND_CHAT %>
    <div class="comment-avatar">
      <%= user_avatar_badge(comment.user, size: 32, klass: "comment-avatar-badge") %>
    </div>

    <div class="comment-main">
      <div class="comment-text">
        <span class="comment-author"><%= display_name_or_anonymous(comment.user) %></span>
        <span class="comment-body"><%= comment.body %></span>
      </div>

      <div class="comment-actions dropdown">
        <button
          class="btn btn-sm btn-link comment-actions-toggle"
          type="button"
          data-bs-toggle="dropdown"
          aria-expanded="false"
          aria-label="コメント操作">
          <i class="bi bi-three-dots"></i>
        </button>

        <ul class="dropdown-menu dropdown-menu-end">
          <li class="comment-menu-item comment-menu-item-cast-normal">
            <button type="button" class="dropdown-item" disabled>
              このコメントを非表示
            </button>
          </li>

          <li class="comment-menu-item comment-menu-item-cast-hidden">
            <button type="button" class="dropdown-item" disabled>
              非表示を解除
            </button>
          </li>

          <li class="comment-menu-item comment-menu-item-viewer-normal">
            <button type="button" class="dropdown-item" disabled>
              このコメントを通報する
            </button>
          </li>
        </ul>
      </div>
    </div>

  <% when Comment::KIND_DRINK %>
    <% metadata = comment.metadata_hash %>
    <% drink_item_id = metadata["drink_item_id"] || metadata[:drink_item_id] %>
    <% drink_item = DrinkItem.find_by(id: drink_item_id) %>

    <div class="comment-avatar">
      <%= user_avatar_badge(comment.user, size: 32, klass: "comment-avatar-badge") %>
    </div>

    <div class="comment-main">
      <div class="comment-text">
        <div class="comment-line">
          <span class="comment-author"><%= display_name_or_anonymous(comment.user) %></span>

          <% if drink_item.present? %>
            <span class="comment-body d-inline-flex align-items-center gap-1">
              <%= drink_item_display(drink_item, icon_size: 32, klass: "comment-drink-item-display") %>
              を送りました
            </span>
          <% else %>
            <span class="comment-body">ドリンクを送りました</span>
          <% end %>
        </div>
      </div>
    </div>

  <% when Comment::KIND_DRINK_CONSUMED %>
    <% metadata = comment.metadata_hash %>
    <% drink_item_id = metadata["drink_item_id"] || metadata[:drink_item_id] %>
    <% drink_item = DrinkItem.find_by(id: drink_item_id) %>

    <div class="comment-avatar">
      <%= user_avatar_badge(comment.user, size: 32, klass: "comment-avatar-badge") %>
    </div>

    <div class="comment-main">
      <div class="comment-text">
        <div class="comment-line">
          <span class="comment-author"><%= display_name_or_anonymous(comment.user) %></span>

          <% if drink_item.present? %>
            <span class="comment-body d-inline-flex align-items-center gap-1">
              <%= drink_item_display(drink_item, icon_size: 32, klass: "comment-drink-item-display") %>
              を消化しました
            </span>
          <% else %>
            <span class="comment-body">ドリンクを消化しました</span>
          <% end %>
        </div>
      </div>
    </div>

  <% else %>
    <div class="comment-avatar">
      <%= user_avatar_badge(comment.user, size: 32, klass: "comment-avatar-badge") %>
    </div>

    <div class="comment-main">
      <div class="comment-text">
        <span class="comment-author"><%= display_name_or_anonymous(comment.user) %></span>
        <span class="comment-body">
          <%= comment.body.presence || "イベントが発生しました" %>
        </span>
      </div>
    </div>
  <% end %>
</div>
```
