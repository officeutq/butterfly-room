今回の実装で、要件との対応はこうなります。

* **Overlay 表示**
  `live_overlay` 内の `.live-overlay-drink-slot` に移設済み

  `turbo_stream_from [@stream_session, :cast_pending_drink_orders]` を Overlay 側に移しただけで、
  replace target の `#cast_pending_drink_orders` は維持

  DOM Overlay だけなので既存方針どおり
