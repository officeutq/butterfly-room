### 2. 店舗管理者エリアの先頭に「ブース管理」カードを追加

`<% if current_user.at_least?(:store_admin) %>` の直後、`店舗設定編集` カードの前に追加してください。

```erb
      <div class="col-12 col-md-6 col-xl-4">
        <%= link_to admin_booths_path, class: "text-decoration-none" do %>
          <div class="card h-100 dashboard-role-card dashboard-role-card-store-admin" data-bs-theme="light">
            <div class="card-body">
              <h5 class="card-title mb-2">ブース管理</h5>
              <p class="card-text mb-0">ブースの作成・紐づけ・アーカイブを管理します</p>
            </div>
          </div>
        <% end %>
      </div>
```

これで `ブース一覧 = cast専用`、`ブース管理 = store_admin以上` に分離できます。
