問題ありません。
その方針なら #339 のスコープにきれいに収まりますし、後続 Issue で各slotの中身を戻していくのもやりやすいです。

今回は **既存UIはすべて `.live-offscreen` に退避**し、Overlay上は

* 映像
* `meta / drink / comment / ops` のデバッグ用色付き枠

だけを見せる形にします。

---

## 最終実装方針

### 1. `live.html.erb`

* `#live_overlay` の中に **Overlay骨格**を作る
* `data-live-grid-debug="1"` を付けて、枠を常時可視化
* `meta / drink / comment / ops` の4slotを配置
* 既存の flash / stream_meta / 操作UI / ドリンク / コメント / 開発用テキスト は **全部 `.live-offscreen`** に移す
* Turbo Stream の購読や target は **DOMとして残す**ので壊さない

### 2. `application.scss`

* `#live_overlay` 自体は `overflow: hidden`
* `.live-overlay-ui` を

  * `meta row = auto（上限あり）`
  * `main row = 1fr`
  * `bottom row = clamp(...)`
    にする
* `main row` と `bottom row` は

  * 左 = 残り
  * 右 = 固定幅 `clamp(...)`
* 各row/slotに `min-width: 0; min-height: 0;`
* debug属性ON時だけ、slotごとに半透明背景 + outline を表示
* このIssueでは slot の中身は空なので、見た目確認用にラベルだけ出す
