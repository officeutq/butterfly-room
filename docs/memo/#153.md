## 実装方針（#153の完了条件に直結する部分だけ）

やることは4点だけです。

1. `GET /system_admin` を追加（`namespace :system_admin do root "dashboard#show" end`）
2. `SystemAdmin::BaseController` を追加（`require_role!(:system_admin)` を共通化）
3. `SystemAdmin::DashboardController#show` を追加（空アクション）
4. `system_admin/dashboard/show.html.erb` を追加（リンク集の枠組み。リンクは仮 `#`）

403は既存どおり `head :forbidden`（`require_role!` がそうなっているため）で統一されます。

## 貼り付け用コード

### 1) `config/routes.rb`（差分：SystemAdmin namespace を追加）

`# --- Store Admin ---` の直前か直後あたりに、以下を追加してください（場所はどこでもOKですが、管理系の並びで置くのが見通し良いです）。

```ruby
  # --- System Admin ---
  namespace :system_admin do
    root "dashboard#show"
  end
```

（※他は変更なし）

---

### 2) `app/controllers/system_admin/base_controller.rb`（新規）

```ruby
# frozen_string_literal: true

module SystemAdmin
  class BaseController < ApplicationController
    before_action -> { require_role!(:system_admin) }
  end
end
```

---

### 3) `app/controllers/system_admin/dashboard_controller.rb`（新規）

```ruby
# frozen_string_literal: true

module SystemAdmin
  class DashboardController < SystemAdmin::BaseController
    def show
    end
  end
end
```

---
