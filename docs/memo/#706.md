## 補足

今回の `stores/show.html.erb` では、ブースカード表示に

```erb
current_user.favorite_booths.exists?(booth_id: booth.id)
```

をその場で使っています。
件数が増えると N+1 っぽくなるので、気になるなら後で controller 側で favorites をまとめて持つようにしてもよいですが、まずはこのままで十分です。

あと、今の `store_params` に `:remove_thumbnail` が入っていないなら、ついでに足しておくとログの unpermitted parameter が消えます。

```ruby
:remove_thumbnail
```

実装したら、見た目の微調整を一緒に詰めましょう。
