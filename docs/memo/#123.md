## まず結論：#123 で直すべき箇所（最小セット）

### A) `StreamSessions::StartService` で stage を束縛する（本題）

* `StreamSession.create!` に `ivs_stage_arn: booth.ivs_stage_arn` を追加
* `booth.ivs_stage_arn.blank?` なら **例外で止める**（設計違反を早期検知）

### B) `IvsParticipantTokensController` から `EnsureIvsStageService` を外す（#122 前提に合わせる）

* #119 で Stage は booth 作成時に出来る前提なので、publisher 側でも **ensure しない**
* `stream_session.ivs_stage_arn.blank?` なら publisher でも `stage_not_bound` で返す

> #123が入れば「作成時に必ず入る」ので、通常ここは通らないはず。
> 逆に通るならデータ不整合なので 409 で正しい。

### C) **DBの unique index が設計と矛盾しているので必ず外す**

`stream_sessions.ivs_stage_arn` に
`unique: true, where: "(ivs_stage_arn IS NOT NULL)"` が付いてます。

Stage を booth 固定で使い回すなら、同じ arn が複数 stream_session に入るので **2回目以降の配信で必ず衝突**します。
→ これは #123 か #122 のどちらかで **必ず撤去**が必要（後回し不可）。

## 実装差分案（コピペしやすい形）

### 1) `StartService` 修正
`app/services/stream_sessions/start_service.rb`
```ruby
# frozen_string_literal: true

module StreamSessions
  class StartService
    class Error < StandardError; end
    class BoothNotOffline < Error; end
    class NotAuthorized < Error; end
    class BoothStageNotBound < Error; end

    def initialize(booth:, actor:)
      @booth = booth
      @actor = actor
    end

    def call
      authorize!

      StreamSession.transaction do
        booth = Booth.lock.find(@booth.id)

        raise BoothNotOffline, "booth is #{booth.status}" unless booth.offline?

        raise BoothStageNotBound, "booth.ivs_stage_arn is blank" if booth.ivs_stage_arn.blank?

        session = StreamSession.create!(
          booth: booth,
          store: booth.store,
          status: :live, # ※ここは現状維持でもOK（後で整理してもよい）
          started_at: Time.current,
          started_by_cast_user: @actor,
          ivs_stage_arn: booth.ivs_stage_arn # ★ここが #123
        )

        booth.update!(
          status: :standby,
          current_stream_session_id: session.id
        )

        session
      end
    end

    private

    def authorize!
      policy = Authorization::BoothPolicy.new(@actor, @booth)
      allowed = policy.cast_operate? # これがあるなら
      raise NotAuthorized unless allowed
    end
  end
end
```
