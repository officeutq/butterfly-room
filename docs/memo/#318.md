## 実装方針（最終）

### ゴール

* `#live_stage` を **映像専用の可視領域**にする
* `video`（normal）と `canvas`（away）を **Stage基準でフィット表示**

  * `video`: `object-fit: contain`
  * `canvas`: intrinsic size（1280×720）を保ったまま `max-width/max-height` で収める（黒帯は Stage 背景）

### 「映像以外は画面外」

* 開発用テキスト、操作UI（離席/復帰/音声）、コメント、ドリンク、stream_meta、flash 等は **画面外**へ退避
* ただし Turbo Stream の target は DOM 上に残す（見えなくてもOK）

  * `#flash_inner`
  * `#cast_pending_drink_orders`（partial内）
  * `#comments`
  * `#comment_form`

### この変更の意図

* `#live_stage` の中は **映像のみ**（`#live_media`）
* それ以外は `.live-offscreen` に退避（DOMは残るので Turbo Stream 破綻しない）
* `video/canvas` の `w-100` / `rounded` / inline background を撤去（Stage責務に寄せる）

```scss
@use "bootstrap/scss/bootstrap" as *;

:root {
  --live-stage-h: 0px;
}

#live_stage {
  height: var(--live-stage-h);
  overflow: hidden;
  position: relative;

  /* 黒帯はUI（Stage背景）として表現する */
  background: #000;
}

#live_media {
  position: absolute;
  inset: 0;
}

/* video は Stage に contain */
#live_media video {
  position: absolute;
  inset: 0;

  width: 100%;
  height: 100%;

  object-fit: contain;
}

  canvas は object-fit が効かないので、
  intrinsic size（width/height属性）を維持して中央に収める

#live_media canvas {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);

  max-width: 100%;
  max-height: 100%;

  width: auto;
  height: auto;

  display: block;
}

/* 映像以外は画面外（DOMは残す） */
.live-offscreen {
  position: absolute;
  left: -100000px;
  top: 0;
  width: 1px;
  height: 1px;
  overflow: hidden;
}

.header-logo {
  height: 24px;
  width: auto;
  display: block;
}
```
