## コピペ実装

### 1) `app/views/stores/show.html.erb`（「店舗プロフィール」セクション追加）

以下の **`<hr class="my-4">` の直後〜「ブース一覧」の前**に差し込んでください。

```erb
  <hr class="my-4">

  <h2 class="h5 mb-3">店舗プロフィール</h2>

  <div class="mb-4">
    <% if @store.thumbnail.attached? %>
      <div class="mb-3">
        <%= image_tag @store.thumbnail, style: "max-width: 240px; height: auto;" %>
      </div>
    <% end %>

    <dl class="row mb-0">
      <dt class="col-sm-3 text-muted">概要</dt>
      <dd class="col-sm-9"><%= @store.description.presence || "-" %></dd>

      <dt class="col-sm-3 text-muted">地域</dt>
      <dd class="col-sm-9"><%= @store.area.presence || "-" %></dd>

      <dt class="col-sm-3 text-muted">業態</dt>
      <dd class="col-sm-9">
        <% if @store.business_type.present? %>
          <%= Store::BUSINESS_TYPE_LABELS[@store.business_type.to_sym] || @store.business_type %>
        <% else %>
          -
        <% end %>
      </dd>
    </dl>
  </div>

  <h2 class="h5 mb-3">ブース一覧</h2>
```

※ 既存の「ブース一覧」見出しは、この追加により **下へ押し出される**形になります（仕様通り）。

### 2) `test/integration/store_show_test.rb`（表示テスト追加）

`test "customer can view store show and see active booths only"` を、以下のように更新してください（**置き換え**でOK）。

```ruby
test "customer can view store show and see active booths only" do
  store = Store.create!(
    name: "store",
    description: "Store description",
    area: "渋谷",
    business_type: :girls_bar
  )

  # thumbnail attached
  store.thumbnail.attach(
    io: File.open(Rails.root.join("test/fixtures/files/thumb.png")),
    filename: "thumb.png",
    content_type: "image/png"
  )

  active_booth = Booth.create!(store: store, name: "active", status: :offline)
  Booth.create!(store: store, name: "archived", status: :offline, archived_at: Time.current)

  cast = User.create!(email: "cast@example.com", password: "password", role: :cast, display_name: "Cast A")
  BoothCast.create!(booth: active_booth, cast_user: cast)

  customer = User.create!(email: "customer@example.com", password: "password", role: :customer)
  sign_in customer, scope: :user

  get store_path(store)
  assert_response :success

  # store basic
  assert_includes @response.body, "store"

  # profile
  assert_includes @response.body, "Store description"
  assert_includes @response.body, "渋谷"
  assert_includes @response.body, "ガールズバー"
  assert_includes @response.body, "<img"

  # booths
  assert_includes @response.body, "active"
  assert_includes @response.body, "Cast A"
  assert_includes @response.body, booth_path(active_booth)

  refute_includes @response.body, "archived"
end
```
