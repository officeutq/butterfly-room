実装方針は **`app/views/dashboard/show.html.erb` のみ変更**です。
`layout_current_booth` を使って各ロールセクション内で current booth を解決し、

* current booth あり → `edit_cast_booth_path(booth)`
* current booth なし → `cast_booths_path`

へ遷移させます。

今回の変更は UI 導線追加だけなので、Epic #285 の範囲内で

* 売上ロジックへの影響：なし
* 配信状態遷移への影響：なし
* current 整合性への影響：なし
* Token API影響：なし

のまま進められます。

* system_admin で current booth あり → 該当 booth の edit に行ける

* cast で current booth あり → 該当 booth の edit に行ける
