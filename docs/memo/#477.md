実装方針です。

1. `app/assets/stylesheets/components/_viewer.scss` を **import 専用の薄い入口** にする
2. 既存 `_viewer.scss` の中身を、責務ごとに 6 partial へ分離する
3. **セレクタ名・順序・値は極力そのまま**にして、デグレ要因を増やさない
4. HTML / JS / `application.scss` は変更しない
5. keyboard mode は comment 単体ではなく **viewer 全体レイアウト制御**なので `_viewer_layout.scss` に置く
6. iPhone Safari の input 補正は comment 責務なので `_viewer_comment.scss` に置く
7. drink panel と drink action row は viewer では連続した導線なので `_viewer_drink.scss` にまとめる

今回の変更対象は以下です。

* 変更: `app/assets/stylesheets/components/_viewer.scss`
