Issue #170 のスコープ（最小導線＋customer専用登録）に合わせて、「既存の stores 登録の型（Form + Service + 薄いController）」を踏襲して実装します。wallet は購入開始時に `find_or_create_by!` で作られるので、登録時には触りません。

---

## 実装方針（最小・既存パターン踏襲）

* **Devise registrations は使わない**（routesの `skip: %i[registrations]` は維持）
* `/sign_up` を **customer登録専用**として追加（GET new / POST create）
* 実装は stores と同じ型に寄せる

  * `Customers::RegistrationForm`（ActiveModel）
  * `Customers::RegisterCustomer`（transactionでUser作成・role固定）
  * `Customers::RegistrationsController`（new/create + sign_in + redirect）
* Home（`/`）は **未ログイン時だけ**「ログイン」「新規アカウント作成」2ボタンにする
* テストは integration 最小（導線表示、登録→role固定、登録後ログイン）

### 2) `app/controllers/customers/registrations_controller.rb`（新規）

```rb
# frozen_string_literal: true

module Customers
  class RegistrationsController < ApplicationController
    skip_before_action :authenticate_user!, raise: false

    def new
      @form = Customers::RegistrationForm.new
    end

    def create
      @form = Customers::RegistrationForm.new(registration_params)

      if @form.save
        sign_in(@form.user) # Devise
        redirect_to root_path, notice: "アカウントを作成しました"
      else
        render :new, status: :unprocessable_entity
      end
    end

    private

    def registration_params
      params.require(:customer_registration).permit(
        :email,
        :password,
        :password_confirmation
      )
    end
  end
end
```

### 3) `app/forms/customers/registration_form.rb`（新規）

```rb
# frozen_string_literal: true

module Customers
  class RegistrationForm
    include ActiveModel::Model
    include ActiveModel::Attributes

    attribute :email, :string
    attribute :password, :string
    attribute :password_confirmation, :string

    attr_reader :user

    validates :email, presence: true
    validates :password, presence: true, confirmation: true
    validates :password_confirmation, presence: true
    validate  :email_must_be_unique

    def save
      return false unless valid?

      result = Customers::RegisterCustomer.call!(
        email: email,
        password: password
      )

      @user = result.user
      true
    rescue ActiveRecord::RecordInvalid => e
      errors.add(:base, e.record.errors.full_messages.join(", "))
      false
    end

    private

    def email_must_be_unique
      return if email.blank?

      if User.exists?(email: email)
        errors.add(:email, "このemailアドレスはすでに使われています")
      end
    end
  end
end
```

### 4) `app/services/customers/register_customer.rb`（新規）

```rb
# frozen_string_literal: true

module Customers
  class RegisterCustomer
    Result = Struct.new(:user, keyword_init: true)

    def self.call!(email:, password:)
      new(email:, password:).call!
    end

    def initialize(email:, password:)
      @email = email
      @password = password
    end

    def call!
      user = nil

      ActiveRecord::Base.transaction do
        user = User.create!(
          email: @email,
          password: @password,
          password_confirmation: @password,
          role: :customer
        )
      end

      Result.new(user: user)
    end
  end
end
```

### 7) `test/integration/customer_sign_up_test.rb`（新規）

既存の integration テスト流儀に合わせます。

```rb
# frozen_string_literal: true

require "test_helper"

class CustomerSignUpTest < ActionDispatch::IntegrationTest
  test "guest home shows login and sign_up links" do
    get root_path
    assert_response :success

    assert_select "a", text: "ログイン", href: new_user_session_path
    assert_select "a", text: "新規アカウント作成", href: sign_up_path
  end

  test "customer can sign up with role fixed and becomes signed in" do
    email = "new_customer@example.com"

    assert_nil User.find_by(email: email)

    assert_difference "User.count", +1 do
      post sign_up_path, params: {
        customer_registration: {
          email: email,
          password: "password",
          password_confirmation: "password"
        }
      }
    end

    user = User.find_by!(email: email)
    assert_equal "customer", user.role

    assert_response :redirect
    follow_redirect!
    assert_response :success
    assert_includes @response.body, "ログイン中: #{email}"
  end
end
```
