* modal を閉じた（`hidden.bs.modal`）タイミングで turbo-frame の中身をクリアして再利用可能にする

### 3) enter が turbo-frame 経由なら「モーダルHTML断片」を返す

* `BoothsController#enter` で `turbo_frame_request?` を見て分岐

  * frame 経由：`render partial: "booths/enter_modal", layout: false`
  * 通常：既存どおり `render :enter`
* 既存の権限制御（store_adminは自店のみ / system_adminは常に）を維持

* **store_admin は「その booth/store を admin できる場合のみ modal」**

  * そうしないと、権限NGのとき `redirect_to booth_path` が “modal frame内遷移” になって崩れるため

### 5) モーダル内ボタンは `_top` 遷移

* 視聴＝GET `/booths/:id`（`booth_path`）
* 配信＝POST `/booths/:id/enter_as_cast`（`enter_as_cast_booth_path`）
* どちらも `data-turbo-frame="_top"` で全画面遷移にする

### 2) app/javascript/controllers/modal_controller.js（新規作成）

```js
import { Controller } from "@hotwired/stimulus"
import { Modal } from "bootstrap"

export default class extends Controller {
  connect() {
    this.modal = null
    this._onHidden = null
  }

  open() {
    // turbo-frame(#modal) の中に modal 断片が入った時だけ開く
    const modalEl = this.element.querySelector(".modal")
    if (!modalEl) return

    // 既に開いている/残骸がある場合は破棄してから作り直す（連打耐性）
    if (this.modal) {
      try { this.modal.hide() } catch (_) {}
      try { this.modal.dispose() } catch (_) {}
      this.modal = null
    }

    this.modal = new Modal(modalEl, {
      backdrop: "static",
      keyboard: true
    })

    // 閉じたら frame を空に戻す（次回のため）
    this._onHidden = () => {
      // dispose してからDOMを消す
      try { this.modal?.dispose() } catch (_) {}
      this.modal = null
      this.element.innerHTML = ""
      this._onHidden = null
    }

    modalEl.addEventListener("hidden.bs.modal", this._onHidden, { once: true })
    this.modal.show()
  }

  disconnect() {
    // Turboのキャッシュ/遷移で disconnect されるケースに備える
    const modalEl = this.element.querySelector(".modal")

    if (modalEl && this._onHidden) {
      try { modalEl.removeEventListener("hidden.bs.modal", this._onHidden) } catch (_) {}
    }

    if (this.modal) {
      try { this.modal.dispose() } catch (_) {}
      this.modal = null
    }
    this._onHidden = null
  }
}
```

    # store_admin（自店なら選択UI / 他店なら視聴）

        if turbo_frame_request?
          render partial: "booths/enter_modal", locals: { booth: @booth }, layout: false, status: :ok

      else
        render :enter, status: :ok
      end
      return
    end

## あなたの質問への回答（汎用的に使える？）

はい、`<turbo-frame id="modal" data-controller="modal" data-action="turbo:frame-load->modal#open">` は **汎用的に使えます**。

ただし “汎用” にするための前提は 2つだけです：

1. `data-turbo-frame="modal"` を付けた遷移は、**modalとして成立する断片HTML**（今回なら `.modal ...`）を返すこと
2. 閉じた後に frame を空に戻す（今回 controller で実施）ことで、**次回も確実に開ける**こと

この2つを守れば、将来ほかの確認ダイアログやフォームも同じ frame で回せます。
