#389 は次の方針で実装するのが最も自然です。

実装方針

* `viewer-overlay-meta-slot` に新しい `viewer-meta-panel` を追加する
* その `viewer-meta-panel` に `presence-poll` を移し、既存 offscreen の

  * `ping_stream_session_presence_path`
  * `presence_summary_stream_session_path`
  * `count` target
    を移植する
* タイトル / 説明は live 側 `_stream_meta_display` と同じ表示方針に揃える

  * `stream_session.title.presence || booth.name`
  * 説明は `present?` のときだけ表示
* `viewer-overlay-meta-slot` に縦スクロール責務を持たせる
* `app/views/booths/_stream_state.html.erb` の live / away 分岐内 offscreen に残っている旧 meta / count / pending drink orders は整理する
* `_ivs_viewer.html.erb` 側の `viewer-legacy-offscreen` は `state` target 用に残す

変更対象

* `app/views/stream_sessions/_ivs_viewer.html.erb`
* `app/views/booths/_stream_state.html.erb`
* `app/assets/stylesheets/components/_viewer.scss`
