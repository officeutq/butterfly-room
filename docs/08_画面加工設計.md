# 08_画面加工設計

## 概要

Butterfly Room の cast/live における配信映像加工のため、
Banuba Web SDK を用いた Beauty effect 資材と、その運用方針を整理する。

本ドキュメントは以下を目的とする。

- Butterfly Room 用 Beauty effect の管理
- Web 用 export 資材の配置ルール整理
- 後続 UI 制御 Issue へ引き渡すパラメータ定義の明文化
- dev 検証画面での確認観点の記録

---

## スコープ

本ドキュメントで扱う対象は以下。

- Banuba Studio で作成する Butterfly Room 用 Beauty effect
- Web export した effect zip
- Rails 検証画面での読込確認
- 後続 UI 実装用のパラメータ一覧

以下は本ドキュメントの直接対象外とする。

- cast/live 画面への本組み込み
- ON/OFF トグル UI 実装
- スライダー UI 実装
- 加工値の永続化
- IVS publish への反映
- 最終 UX 調整

---

## 構成方針

### 1. effect は 1 本の Beauty base にまとめる

Phase1 では複数の effect zip を切り替える方式ではなく、
1 本の `beauty_base.zip` に必要な Beauty / face reshape 系機能をまとめる。

理由:

- UI 側で ON/OFF / スライダー制御しやすい
- effect 資材の管理が単純
- cast/live 組み込み時の扱いが楽
- 後続実装で effect 切り替えロジックを避けられる

### 2. 初期値は弱めにする

初期状態は「やりすぎ感」を避けるため、弱め〜中弱程度を基本とする。

### 3. OFF 相当は 0.0 に統一する

後続 UI 制御を簡単にするため、
各パラメータの無効状態は 0.0 で統一する。

## パラメータ定義

### 方針

パラメータ名は、後続 UI 実装時に人間が意味を理解しやすい名前で管理する。

### 一覧

- `skin_smoothing`
  - 意味: 美肌補正
  - デフォルト値: `0.20`
  - 推奨レンジ: `0.0..0.6`
  - OFF 相当: `0.0`

- `face_contour`
  - 意味: 輪郭補正
  - デフォルト値: `0.10`
  - 推奨レンジ: `0.0..0.3`
  - OFF 相当: `0.0`

- `eye_enlarge`
  - 意味: 目の補正
  - デフォルト値: `0.08`
  - 推奨レンジ: `0.0..0.25`
  - OFF 相当: `0.0`

- `nose_reshape`
  - 意味: 鼻の補正
  - デフォルト値: `0.06`
  - 推奨レンジ: `0.0..0.2`
  - OFF 相当: `0.0`

- `lip_reshape`
  - 意味: 口の補正
  - デフォルト値: `0.05`
  - 推奨レンジ: `0.0..0.2`
  - OFF 相当: `0.0`

## Banuba Studio での作成手順

### 1. 新規 Beauty base を作成する

Banuba Studio で Butterfly Room 用の Beauty effect を新規作成する。

### 2. 加工項目を追加する

最低限、以下を含める。

- 美肌
- 輪郭
- 目
- 鼻
- 口

### 3. 初期値を設定する

配信用途のため、強すぎない自然な補正を初期値とする。
以下は Banuba Studio の Morphing / Retouch パラメータの初期値。

#### Retouch（美肌）

* Eyes Whitening: **0.20**
* Eyes Flare: **0.10**
* Teeth Whitening: **0.10**
* Skin Smoothing Strength: **0.35**

#### Morphing → Face

* Narrowing: **0.12**
* V Shape: **0.03**
* Cheekbones Narrowing: **0.02**
* Cheeks Narrowing: **0.04**
* Jaw Narrowing: **0.04**
* Chin Shortening: **0.01**
* Chin Narrowing: **0.03**
* Cheeks Jaw Narrowing: **0.02**

#### Morphing → Eyes

* Rounding: **0.03**
* Enlargement: **0.07**
* Height: **0.03**
* Squint: **-0.02**
* Lower Eyelid Position: **0.01**
* Eyelid Upper: **0.02**
* Eyelid Lower: **0.01**

#### Morphing → Nose

* Width: **-0.05**
* Length: **0.02**
* Tip Width: **-0.03**

#### Morphing → Lips

* Size: **0.02**

※ Eyebrows は初期値では変更しない


### 4. 極端値を確認する

少なくとも以下を確認する。

- デフォルト値で違和感が強すぎない
- 最小値 0.0 で補正が外れる
- 最大寄りで顔形状が破綻しない
- 顔移動時に追従が外れない

### 5. Web 用に export する

Banuba Studio から Web 用 effect として export し、
出力 zip を `beauty_base.zip` として扱う。

### 6. Rails へ配置する

以下へ配置する。

- `public/banuba/effects/beauty_base.zip`

## dev 検証画面での確認観点

### 起動確認

- カメラ権限が取得できる
- 映像が表示される
- `beauty_base.zip` が読み込まれる
- 顔追従が継続して動作する

### Beauty 確認

- 美肌が強すぎない
- 輪郭補正が不自然でない
- 目の補正が違和感なく追従する
- 鼻補正が破綻しない
- 口補正が破綻しない

### 安定性確認

- 数分放置でも console error が出ない
- 停止 → 起動 を繰り返しても壊れない
- Turbo 遷移後にカメラが残らない
