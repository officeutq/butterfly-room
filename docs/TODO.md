### お気に入り画面
* [x] 切り替えタブをヘッダー内に移動する。検索フォームを追加する

### チュートリアル
* [x] 招待URLを共有、にフォーカスするとき、スクロールするようにする

### 店舗新規作成
* [x] ブースを同時に作らないようにする

### ブース作成・編集画面
* [x] ここでブースにキャストを紐づけられるようにする。ただし、未設定の場合に限る。必須ではない。ブース管理画面を参考にして、フォームにドロップダウンを追加する。

### ブース編集画面
* [x] 紐づいているキャストを表示する。

### ブース管理画面
* [x] タイトルをブース管理にする。
* [x] キャスト紐づけロジックを削除する。(ブース編集からのみにする)
* [x] 現在のキャスト→所属キャストにして、アイコンを表示させる

### ブース情報画面
* [ ] ブースURLをコピーボタンの廃止。
* [ ] 所属キャスト名の追加。ブース詳細画面に寄せる

### キャスト一覧画面
* [ ] キャストのメールアドレスの表示を廃止。

### ドリンクメニュー画面
* [ ] 0ptのドリンクを作成可能にする。色も考慮する。

### 清算・振り込み設定画面
* [ ] 振込口座設定に名前を変える。
* [ ] 入力カードはライトモードにする。
* [ ] 郵貯対応にする。
* [ ] デザインは電話番号認証画面に寄せる。

### 清算（予定・履歴）画面
* [ ] 入力カードをライトモードにする。
* [ ] ダッシュボードへ戻るボタンの廃止

### 電話番号認証画面
* [ ] 表示が+81表示になっているので、日本風に置き換えて表示する。

### プロフィール編集画面
* [ ] 表示名の注意書きを変更する。未入力の場合はemailが表示されます。となっているが、ななしさんにする。

### コメント
* [ ] 配信者がコメントを通報できるようにする。
* [ ] コメントの・・・を押下したときの表示の崩れを修正する→ポップオーバーにする

### ブース詳細画面
* [x] キャストにお気に入りボタンがないのでつける。
* [x] ブースのお気に入りボタンをブース名のすぐ後に移動する。

### castのブース選択モーダル
* [ ] IDの表示を削除

### 視聴画面
* [ ] 配信終了時のフラッシュが残っている場合、次の配信開始時に消す
